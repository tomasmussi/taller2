INSTALACION HEROKU + NODE.JS + POSTGRESQL
1- Instalar Heroku https://devcenter.heroku.com/toolbelt-downloads/debian
2- Instalar Postgresql sudo apt install postgresql
3- Descargar el proyecto del GitHub
4- Ingresar a la carpeta taller2/SharedServer y ejecutar heroku local database
5- Luego ejecutar "heroku local web"
