Linkedun
Manual de instalación de servidores
  


________________


Objetivo
Describir, paso a paso, la instalación de los servidores que se comunican con la app de Android de Linkedun.


Requisitos
Para la construcción y su posterior ejecución de los servidores App Server y Shared Server es necesaria tener descargado el proyecto de la siguiente URL:
* https://github.com/tomasmussi/taller2.git
Puede realizarlo haciendo:
        git clone https://github.com/tomasmussi/taller2.git
deberá tener previamente instalado git en su sistema operativo. También es necesario tener instalado Docker en su sistema operativo [1].
Instalación App server


Posicionarse en el directorio principal de la aplicación ( taller2/server/ ). En el mismo existe un archivo llamado Dockerfile el cual contiene los comandos necesarios para llevar a cabo la dockerizacion del App Server.
Ejecutar el siguiente comando:
* docker build -t appserver .
Luego de la construcción ejecutar:
* docker run -p 0.0.0.0:8080:8080 appserver
Visualizará la siguiente lista de servicios:

Instalación Shared server


Posicionarse dentro de la carpeta ( taller2/SharedServerHeroku/database/ ). Crear una carpeta llamada “logs”. En el mismo directorio mencionado existe un archivo llamado Dockerfile el cual contiene los comandos necesarios para llevar a cabo la dockerizacion del Shared Server.
Ejecutar el siguiente comando:
* docker build -t sharedserver-database .
Esto generará la dockerizacion de la base de datos, luego hay que posicionarse en la carpeta principal ( taller2/SharedServerHeroku/ ) y ejecutar el siguiente comando:
* docker-compose up
Este comando dockerizara NodeJs y unirá las dos aplicaciones (PostgreSQL y NodeJs) quedando el servidor operativo. Para detener la ejecución del servidor ejecutar Control + C.
